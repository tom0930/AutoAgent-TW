# Project: AutoAgent-TW
Version: v3.5.2
Status: Active (Phase 164 - Subagent Context Isolation Completed)
Description: 專注於全自動、無感、具備高韌性 (Resilience) 的高級 AI 系統與知識庫自動化管道。
Type: Python CLI & Autonomous Agent Workspace

## 📝 專案說明 (Project Description)
AutoAgent-TW 是一個高度自動化、具備背景排程與防呆修復 (Self-Repair) 能力的自主代理開發環境。
v3.x 之開發核心為「多代理編排與語境隔離 (Multi-Agent Orchestration & Context Isolation)」，將子代理 (Subagent) 放置於受限且優化的環境中執行專家任務。

## 🎯 核心目標 (Core Objectives)
1. 將原本脆弱的單次腳本執行強化為能在惡劣網路環境與突發異常下存活的高韌性系統。
2. 透過 Subagent Isolation 與 RTK Pruning 解決 Agent Overload 問題，節省 90% 以上的 Token 消耗。
3. 實作意圖式工具發現 (Proactive Tool Discovery)，讓 Agent 能自主擴展功能邊界。
