# AI Harness Gateway дҪҝз”ЁжүӢеҶҠ

## еҝ«йҖҹй–Ӣе§Ӣ

```bash
# е•ҹеӢ• Gateway
python src/core/harness_gateway.py start

# жҹҘзңӢзӢҖж…Ӣ
python src/core/harness_gateway.py status

# еҒңжӯў Gateway
python src/core/harness_gateway.py stop

# еүҚеҸ°еҹ·иЎҢ
python src/core/harness_gateway.py run
```
