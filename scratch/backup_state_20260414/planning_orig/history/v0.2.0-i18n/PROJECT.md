# Project: AutoAgent-TW Multi-Language (i18n) Support
# Issue Reference: #1

## Core Objectives
1.  Enable multi-language support (English and Traditional Chinese) for all `aa-` commands.
2.  Modify the workflow files (`.md` in `_agents/workflows`) to support dynamic language display (or provide dual-language descriptions).
3.  Ensure the Antigravity Skills reflect these changes.

## Technology Stack
- Markdown for workflows
- Skill metadata for the IDE UI
- Configuration-based language selector (`.planning/config.json`)

## Initial Roadmap
- Phase 1: Research i18n strategies for Antigravity workflows.
- Phase 2: Implementation of bilingual descriptions in all 14 workflows.
- Phase 3: Validation and QA.
